# GMG-JS
GMG JS common cross-project classes &amp; functions

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install https://github.com/GmgSante/GMG-JS
$ npm rm gmg-js

```

## Use 

```sh
const Logger = require('gmg-js').Logger;

let logger = new Logger();
```
